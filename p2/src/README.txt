Package:
Python: 2.7.10

1. Put train, test-private, test-public directories into same folder with code files
2. Open terminal and go to the folder where data and code in
3. Run code: python main.py
4. F scores for different algorithms with different taggers will show in terminal and detection results will be in corresponding csv files:
    file name: [algorithm]_[tagger]_[phrase/sentence]_results.csv



crf model should be tested directly on crf_model.py instead of running from main.py
